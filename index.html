Projeto: Sistema de Agendamento

Este projeto consiste em um sistema de agendamento desenvolvido em Java, criado como parte do Trabalho de Conclusão de Curso. Nosso objetivo foi desenvolver um sistema que atendesse às necessidades específicas do salão de estética Perpétua. Após entrevistas e questionários, levantamos informações detalhadas para garantir que todas as demandas dos stakeholders fossem plenamente atendidas.

Projeto: Sistema de Help Desk

Este projeto consiste em um sistema de Help Desk desenvolvido utilizando Java e Angular, com o objetivo de aprimorar meus conhecimentos em tecnologias modernas e abordagens de desenvolvimento web. A proposta foi criar uma solução funcional que permitisse gerenciar solicitações e atendimentos, explorando funcionalidades como autenticação, controle de acesso e persistência de dados.

As tecnologias utilizadas incluem Angular para a interface do usuário, Spring Security para segurança e autenticação, Spring Data para gerenciamento de dados e MySQL como banco de dados. Este projeto também foi uma oportunidade de aplicar conceitos de integração entre frontend e backend, além de seguir boas práticas de desenvolvimento. Quer saber mais sobre o projeto?
